# URL Path는 숫자 접두사를 그대로 유지한다

초기 설계에서는 `02-basics/01-variables.md` → `/basics/variables` 처럼 `NN-` 접두사를 제거해 깔끔한 URL을 만들었다. 그러나 마크다운 파일 내 상대 링크(`[함수](./02-functions.md)`)가 실제 파일 경로를 기반으로 작성되기 때문에, 서버의 URL이 다르면 링크가 404로 끊어진다. `.md` 확장자만 제거하고 접두사는 유지하는 방식(`/02-basics/01-variables`)으로 변경하면 상대 경로 링크가 경로 변환 없이 그대로 동작한다. URL이 다소 길어지는 단점이 있지만, 링크 신뢰성이 더 중요하다.

## Considered Options

- **접두사 제거** (`/basics/variables`) — 깔끔하지만 상대 링크와 불일치. HTML 후처리로 모든 링크를 재작성해야 하므로 변환 오류 가능성이 있다.
- **접두사 유지** (`/02-basics/01-variables`) — 채택. `.md` 제거만 하면 되므로 상대 링크가 자연스럽게 동작한다.
